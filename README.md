PlaceBridge – Smart Placement Portal

Project Description

PlaceBridge is a web-based placement management system designed to simplify the campus recruitment process.
It allows students to check eligibility for companies, apply for specific roles, and track their application status in a single dashboard.

This prototype demonstrates a centralized placement portal where companies can define roles and students can apply based on CGPA, skills, and backlog criteria.

---

Problem Statement

Students often miss placement opportunities due to fragmented communication, manual eligibility checking, and lack of real-time updates.
Existing systems do not provide a simple way to track applications and eligibility in one place.

PlaceBridge solves this problem by providing a unified and easy-to-use placement application portal.

---

Features

- Student profile with CGPA, skills, and backlogs
- Smart eligibility checker
- Multiple companies with multiple roles
- Apply for specific role
- Application tracker
- Real-time alerts and notifications (prototype)
- Dashboard with company list
- Clean and responsive UI
